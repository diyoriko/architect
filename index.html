How it works

4 проекта: SAMI (fitness community bot), Hunter (job aggregator bot) — Railway 24/7, автодеплой из main. Sugata Jyotish (vedic astrology) — Vercel. Portfolio (diyor.design) — GitHub Pages.

Ежедневно 09:30 МСК — на Mac запускаются 2 стратега (SAMI + Hunter). Каждый через Claude анализирует свой проект: бэклог, метрики, код — и шлёт отчёт с рекомендациями в Telegram. Предложения стратега попадают в бэклог проекта.

По воскресеньям 10:00 МСК — Architect делает глубокий кросс-проектный ревью: health check всех сервисов, coverage, dependencies, code churn, паттерны работы (burnout risk, context switching, commit hours). Пишет задачи в свой бэклог, обновляет этот дашборд, создаёт Google Calendar event.

При каждом коммите — pre-commit hook проверяет TypeScript типы (tsc --noEmit). Если ошибки — коммит блокируется.

При каждом push/PR — GitHub Actions CI запускает typecheck + все тесты (vitest). Coverage не должен падать ниже 20%. При PR — CodeRabbit (AI) делает code review и пишет замечания в бэклог проекта.

Каждую ночь 03:00 МСК — GitHub Actions бэкапит SQLite базы обоих ботов из Railway в GitHub Artifacts (30 дней хранения).

22:30 МСК — Mac notification: «Пора заканчивать. Сохрани сессию (/save) и закрывай.»

Claude Max ($200/mo) покрывает всю интерактивную работу + стратегов + архитектора (через Mac launchd, бесплатно). API key ($5 balance) тратится только на GitHub Actions и Hunter cover letters.
