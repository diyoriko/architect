Как всё работает

Два бота (SAMI и Hunter) крутятся 24/7 на Railway — автодеплой при пуше в main.

Каждое утро в 09:30 МСК на Mac запускаются стратеги — они анализируют проект через Claude API и шлют отчёт в Telegram. По воскресеньям в 10:00 запускается Architect — делает глубокий недельный ревью всех проектов, пишет задачи в бэклог.

При каждом пуше CI проверяет типы (tsc) и запускает тесты (vitest). При PR — CodeRabbit делает ревью и пишет замечания в бэклог проекта. Перед коммитом pre-commit hook не даёт закоммитить с ошибками типов.

Каждую ночь в 03:00 GitHub Actions делает бэкап SQLite баз обоих ботов.
